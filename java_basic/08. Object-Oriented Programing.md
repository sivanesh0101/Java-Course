## 1. Classes and Objects

###  What is a Class?
- A **class** is a blueprint or template for creating objects.
- It defines **fields (variables)** and **methods (functions)** that represent the state and behavior of an object.

###  What is an Object?
- An **object** is an instance of a class.
- Each object has its own copy of **instance variables**, but shares class-level members (static).

###  Example:
```java
class Car {
    String brand;   // field
    int speed;      // field

    void drive() {  // method
        System.out.println(brand + " is driving at " + speed + " km/h");
    }
}

public class Main {
    public static void main(String[] args) {
        Car car1 = new Car();    // object creation
        car1.brand = "Tesla";
        car1.speed = 120;
        car1.drive();            // Tesla is driving at 120 km/h

        Car car2 = new Car();    // another object
        car2.brand = "BMW";
        car2.speed = 100;
        car2.drive();            // BMW is driving at 100 km/h
    }
}
```

## 2. Methods (Static & Non-Static)

###  Non-Static Methods
- Belong to an **object**.
- Require object creation to be called.
- Can access both instance and static variables.

```java
class Car {
    String brand;
    void showBrand() {
        System.out.println("Brand: " + brand);
    }
}

public class Main {
    public static void main(String[] args) {
        Car car = new Car();
        car.brand = "Tesla";
        car.showBrand(); // Brand: Tesla
    }
}
```

###  Static Methods
- Belong to the **class** itself, not to objects.
- Can be called directly with `ClassName.methodName()`.
- Cannot directly access non-static members.

```java
class MathUtil {
    static int square(int x) {
        return x * x;
    }
}

public class Main {
    public static void main(String[] args) {
        int result = MathUtil.square(5); // 25
        System.out.println(result);
    }
}
```

## 3. Constructors (Default & Parameterized)

###  What is a Constructor?
- A special method used to **initialize objects**.
- Has the same name as the class.
- No return type (not even `void`).
- Automatically called when you create an object.

###  Default Constructor
- Provided by Java if you donâ€™t write any constructor.
- Initializes variables to default values.

```java
class Car {
    String brand;
    int speed;

    Car() { // default constructor
        brand = "Unknown";
        speed = 0;
    }

    void show() {
        System.out.println(brand + " : " + speed + " km/h");
    }
}

public class Main {
    public static void main(String[] args) {
        Car c = new Car();
        c.show(); // Unknown : 0 km/h
    }
}
```

###  Parameterized Constructor
- Accepts arguments to initialize variables with custom values.

```java
class Car {
    String brand;
    int speed;

    Car(String b, int s) { // parameterized constructor
        brand = b;
        speed = s;
    }

    void show() {
        System.out.println(brand + " : " + speed + " km/h");
    }
}

public class Main {
    public static void main(String[] args) {
        Car car1 = new Car("Tesla", 120);
        Car car2 = new Car("BMW", 100);

        car1.show(); // Tesla : 120 km/h
        car2.show(); // BMW : 100 km/h
    }
}
```

## 4. `this` Keyword

###  What is `this`?
- Refers to the **current object**.
- Helps to differentiate between instance variables and local variables with the same name.
- Can also be used to:
  - Call another constructor (`this()`).
  - Pass current object as argument.

###  Example:
```java
class Car {
    String brand;
    int speed;

    Car(String brand, int speed) {
        this.brand = brand;   // differentiate local vs instance
        this.speed = speed;
    }

    void display() {
        System.out.println(this.brand + " drives at " + this.speed + " km/h");
    }
}

public class Main {
    public static void main(String[] args) {
        Car car1 = new Car("Tesla", 120);
        car1.display(); // Tesla drives at 120 km/h
    }
}
```

## 5. Method Overloading (Compile-Time Polymorphism)

###  What is Overloading?
- Multiple methods with the **same name** but **different parameters**.
- Decided at **compile time**.
- Improves code readability.

###  Example:
```java
class MathUtil {
    int add(int a, int b) {
        return a + b;
    }

    double add(double a, double b) {
        return a + b;
    }

    int add(int a, int b, int c) {
        return a + b + c;
    }
}

public class Main {
    public static void main(String[] args) {
        MathUtil obj = new MathUtil();
        System.out.println(obj.add(2, 3));       // 5
        System.out.println(obj.add(2.5, 3.5));   // 6.0
        System.out.println(obj.add(1, 2, 3));    // 6
    }
}
```

## 6. Access Modifiers

###  Types of Access Modifiers
- **public** â†’ accessible from anywhere.
- **private** â†’ accessible only within the same class.
- **protected** â†’ accessible in the same package + subclasses.
- **default (no modifier)** â†’ accessible only within the same package.

###  Example:
```java
class Example {
    public int a = 10;       // anywhere
    private int b = 20;      // only inside this class
    protected int c = 30;    // same package + subclasses
    int d = 40;              // default (package-private)

    void show() {
        System.out.println(a + " " + b + " " + c + " " + d);
    }
}

public class Main {
    public static void main(String[] args) {
        Example e = new Example();
        e.show();  // Can access all here because we are in the same class
    }
}
```

## 7. Encapsulation (Getters & Setters)

###  What is Encapsulation?
- Definition:
It means binding data (fields/variables) and methods (functions) that operate on that data into a single unit (class), and restricting direct access to some of the object's components.
- Making variables **private** and accessing them via **getters & setters**.
- Ensures data security.

###  Example:
```java

class BankAccount {
    public double balance; //  Directly accessible
}

public class Main {
    public static void main(String[] args) {
        BankAccount acc = new BankAccount();
        acc.balance = -5000;
        System.out.println(acc.balance);
    }
}
```

```java
class BankAccount {
    private double balance = 1000;

    public void setBalance(double amount) {
        balance = amount;
    }

        return balance;
```
