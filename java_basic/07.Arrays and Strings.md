### Arrays in Java

## What is Array

an array is a container object that holds a fixed number of elements of the same data type. Arrays are used to store multiple values in a single variable, instead of declaring separate variables for each value.

![image](https://github.com/user-attachments/assets/1b3a7205-9fa3-443a-b0f1-19f51dd474bc)

To declare an array, define the variable type with square brackets

#### Systax

      String[] cars;

We have now declared a variable that holds an array of strings. To insert values to it, you can place the values in a comma-separated list, inside curly braces

### syntax
    
   String[] cars = {"Volvo", "BMW", "Ford", "Mazda"};

   To create an array of integers

  int[] myNum = {10, 20, 30, 40};

### Key Features of Arrays:

1.Fixed size: Once created, its size can't be changed.

2.Zero-based index: The first element is at index 0.

3.Homogeneous: All elements must be of the same type.

4.Indexed access: You can quickly access any element using its index.

### Three Dimensional Array

public class ThreeDArrayExample {
    public static void main(String[] args) {
        
        // 3D Array with 2 layers, 2 rows, 2 columns
        int[][][] arr = {
            { {1, 2}, {3, 4} },   // Layer 0
            { {5, 6}, {7, 8} }    // Layer 1
        };

        // Print all elements layer by layer
        for (int layer = 0; layer < arr.length; layer++) {
            System.out.println("Layer " + layer + ":");
            
            for (int row = 0; row < arr[layer].length; row++) {
                for (int col = 0; col < arr[layer][row].length; col++) {
                    System.out.print(arr[layer][row][col] + " ");
                }
                System.out.println(); // new line after row
            }
            System.out.println(); // space between layers
        }
    }
}

# Java Arrays Utility Methods (`java.util.Arrays`)

### 1. toString()
Converts a 1D array into a readable string.
```java
int[] arr = {10, 20, 30};
System.out.println(Arrays.toString(arr)); 
// [10, 20, 30]
```

---

### 2. deepToString()
Converts multi-dimensional arrays into a readable string.
```java
int[][] matrix = {{1,2}, {3,4}};
System.out.println(Arrays.deepToString(matrix));  
// [[1, 2], [3, 4]]
```

---

### 3. sort()
Sorts array in ascending order.
```java
int[] arr = {40, 10, 30, 20};
Arrays.sort(arr);
System.out.println(Arrays.toString(arr));
// [10, 20, 30, 40]
```

---

### 4. parallelSort() (Java 8+)
Sorts using multiple CPU cores (faster for large arrays).
```java
int[] arr = {40, 10, 30, 20};
Arrays.parallelSort(arr);
System.out.println(Arrays.toString(arr));
// [10, 20, 30, 40]
```

---

### 5. binarySearch()
Searches for an element (array must be sorted).
```java
int[] arr = {10, 20, 30, 40};
int index = Arrays.binarySearch(arr, 30);
System.out.println(index);  // 2
```

---

### 6. copyOf()
Copies array into a new one of given length.
```java
int[] arr = {10, 20, 30};
int[] newArr = Arrays.copyOf(arr, 5);
System.out.println(Arrays.toString(newArr));
// [10, 20, 30, 0, 0]
```

---

### 7. copyOfRange()
Copies a specific range from array.
```java
int[] arr = {10, 20, 30, 40, 50};
int[] part = Arrays.copyOfRange(arr, 1, 4);
System.out.println(Arrays.toString(part));
// [20, 30, 40]
```

---

### 8. equals()
Checks if two arrays are equal.
```java
int[] a = {1, 2, 3};
int[] b = {1, 2, 3};
System.out.println(Arrays.equals(a, b));  // true
```

---

### 9. deepEquals()
Checks equality of multi-dimensional arrays.
```java
int[][] a = {{1,2}, {3,4}};
int[][] b = {{1,2}, {3,4}};
System.out.println(Arrays.deepEquals(a, b)); // true
```

---

### 10. fill()
Fills entire array with a given value.
```java
int[] arr = new int[5];
Arrays.fill(arr, 7);
System.out.println(Arrays.toString(arr));
// [7, 7, 7, 7, 7]
```

## 1. `length()`

## 2. `charAt(index)`

Returns the character at a given index (0-based).

```java
public class StringCharAt {
    public static void main(String[] args) {
        String text = "Java";
        System.out.println("Char at 0: " + text.charAt(0)); // 'J'
        System.out.println("Char at 2: " + text.charAt(2)); // 'v'
    }
}
```

**Output:**

```
Char at 0: J
Char at 2: v
```

## 3. `toUpperCase()` and `toLowerCase()`

Converts the string to uppercase or lowercase.

**Output:**

## 4. `substring(start, end)`

Extracts part of the string.

```java
public class StringSubstring {

        System.out.println("From index 3: " + text.substring(3));     // "gramming"
        System.out.println("From 0 to 6: " + text.substring(0, 6));  // "Progra"
    }
}
```

**Output:**

```
From index 3: gramming
From 0 to 6: Progra
```

## 5. StringBuilder & StringBuffer

**Explanation:**  
- **StringBuilder** – Mutable, faster (not thread-safe)
- **StringBuffer** – Mutable, thread-safe (slower)

**Example:**
```java
public class StringBuilderExample {
    public static void main(String[] args) {
        StringBuilder sb = new StringBuilder("Hello");
        sb.append(" World");
        System.out.println(sb);
        
        sb.insert(6, "Java ");
        System.out.println(sb);
        
        sb.reverse();
        System.out.println(sb);
    }
}
```

## 6. String Immutability Concept

**Explanation:**  
Once a `String` is created, it **cannot be modified**. Any change creates a **new String object** in memory.

**Example:**
```java
public class Immutability {
    public static void main(String[] args) {
        String s1 = "Java";
        String s2 = s1;                 // same reference now
        s1 = s1 + " Rocks";
        System.out.println("s1 == s2 ? " + (s1 == s2)); 

        s2 = s1;             // NEW object created for s1

        System.out.println("s1: " + s1);           
        System.out.println("s2: " + s2);           
        System.out.println("s1 == s2 ? " + (s1 == s2)); 
        System.out.println("s1.equals(s2)? " + s1.equals(s2));
    }
}

```
