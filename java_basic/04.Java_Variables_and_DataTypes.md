## ğŸ§  What is a Variable?
A **variable** is a **name** that stores **data** in your program. Think of it like a **container** or **box** with a label on it.

```java
int age = 25;
```

Here:
- `int` is the **data type** (tells Java what kind of data it is).
- `age` is the **variable name**.
- `25` is the **value** stored in that variable.

## ğŸ§ª Why Do We Need Variables?
- To **store** data (like numbers, names, results)
- To **change** and **reuse** data in the program
- To **perform operations** (like calculations)

# 3. Variables & Constants

**Explanation:**
- **Variable:** A storage location with a name.
- **Constant:** Declared with `final` and cannot be changed.

**Example:**
```java
public class VariablesAndConstants {
    public static void main(String[] args) {
        int marks = 85;  // variable
        final double PI = 3.14159;  // constant

        System.out.println("Marks: " + marks);
        System.out.println("PI: " + PI);

        marks = 90; // allowed
        // PI = 3.14; // âŒ error, cannot change constant
    }
}
```

## 4. Type Casting

### Types:
- **Implicit Casting (Widening):** Smaller type â†’ Larger type (automatic)
- **Explicit Casting (Narrowing):** Larger type â†’ Smaller type (manual)

**Example:**
```java
public class TypeCastingExample {
    public static void main(String[] args) {
        int num = 100;
        double doubleNum = num; // implicit casting
        System.out.println("Double: " + doubleNum);

        double price = 99.99;
        int intPrice = (int) price; // explicit casting
        System.out.println("Int Price: " + intPrice);
    }
}
```
