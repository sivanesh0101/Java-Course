## Questions

1. Write a Java program that takes an integer input n and prints its multiplication table up to 10 using a for loop.
2. Write a program that accepts a number from the user and checks if it is even or odd using an if-else statement.
3. Write a Java program that stores 5 numbers in an array and prints the largest number using a for loop and an if condition.
4. Write a program that stores 10 integers in an array. Using a for loop and if-else conditions, count how many numbers are even and how many are odd.
