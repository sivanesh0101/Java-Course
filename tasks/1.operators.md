# Java Practice Challenges Solutions

## 1. Area of Rectangle, Circle, Triangle
```java
public class AreaCalculator {
    public static void main(String[] args) {
        final double PI = 3.14159; // constant

        // Rectangle
        double length = 5;
        double width = 3;
        double rectangleArea = length * width;
        System.out.println("Area of Rectangle: " + rectangleArea);

        // Circle
        double radius = 4;
        double circleArea = PI * radius * radius;
        System.out.println("Area of Circle: " + circleArea);

        // Triangle
        double base = 6;
        double height = 8;
        double triangleArea = 0.5 * base * height;
        System.out.println("Area of Triangle: " + triangleArea);
    }
}
```

## 2. Celsius to Fahrenheit Converter
```java
public class TemperatureConverter {
    public static void main(String[] args) {
        double celsius = 37;
        double fahrenheit = (celsius * 9/5) + 32;
        System.out.println(celsius + "°C = " + fahrenheit + "°F");
    }
}
```

## 3. Student Grade Calculator (Ternary Operators)
```java
public class GradeCalculator {
    public static void main(String[] args) {
        int marks = 85;
        
        String grade = (marks >= 90) ? "A+" :
                       (marks >= 80) ? "A" :
                       (marks >= 70) ? "B" :
                       (marks >= 60) ? "C" : "Fail";
        
        System.out.println("Marks: " + marks);
        System.out.println("Grade: " + grade);
    }
}
```

## 4. Swap Two Numbers Without Third Variable
```java
public class SwapNumbers {
    public static void main(String[] args) {
        int a = 10, b = 20;

        System.out.println("Before Swap: a = " + a + ", b = " + b);

        a = a + b; // a = 30
        b = a - b; // b = 10
        a = a - b; // a = 20

        System.out.println("After Swap: a = " + a + ", b = " + b);
    }
}
```

## 5. Currency Converter (USD ↔ INR)
```java
public class CurrencyConverter {
    public static void main(String[] args) {
        final double USD_TO_INR = 83.2; // Example rate

        // USD to INR
        double usd = 50;
        double inr = usd * USD_TO_INR;
        System.out.println(usd + " USD = " + inr + " INR");

        // INR to USD
        double inrAmount = 8320;
        double usdAmount = inrAmount / USD_TO_INR;
        System.out.println(inrAmount + " INR = " + usdAmount + " USD");
    }
}
```
